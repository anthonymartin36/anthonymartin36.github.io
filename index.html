What is grid based design?

This is a method of leveraging the capacity of CSS and HTML to display content relative to a webpage. For example say you have 3 stories of content that you wish to display on page vertically in parallel on a webpage. The most suitable method that can be used would be grid approach using CSS Row's and Column's. This would allow content to be displayed in such a way. It is also important to note you can also use HTML tables, however this would not be easily responsive. More..
