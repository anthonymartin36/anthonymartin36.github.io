T6 JavaScript Language

    JavaScript syntax and how they are used.
  • Parentheses () : In software parentheses () are used in loop and functions. What I mean by this is what is entered into a the parentheses () is considered a condition, a condition which is either to be processed or repeated. The same goes for a loop provided the context of what is being asked is set to be true the process ot the loop will repeat itself.
  • Brackets [] : There are a number of different methods of storing information with code. Probable the most preferent would be the array. Now an array is a method of storing a number of different values in the context of one declarable variable. What I mean by this is that the different values in the array can be referenced, amended and other uses by using the Brackets [].
  • Braces {} : Software development is very logical, code writen needs to be clean and specific and have good structure. One example of this is to use Braces {}. Braces allow for logic to be refined to specific instances, say functions, loops or objects. In functions and loops, the body of the logical processes to produce a specific result are carried out within the Braces {}. In Objects much like the arrays above the we can store values and values of values each with its own referencable property. Core of this message is that like sentence structure in paragraphs or stanzas, Braces{} are the coding equivalents.
  • Single Quotes '' : When declaring text to the screen or using any literal term with Javascript, the contents within a '' can be produced to the screen or other used in other contexts.
  • Double quotes "" : Similar to the single quotes above you can contain text within the code. However, the code can be more malleable when displaying the text within the Quotes "" on screen.
