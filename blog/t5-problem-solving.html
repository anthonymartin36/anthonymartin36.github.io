T5 Problem Solving

What was the problem?
The problem was to use the code that we had learnt to create a game to edit 9 different areas in a box and show then filled on a screen in real time. Then tell an end user number of areas that had been filled. Also the boxes that were being filled were of different colors some with no colors. This all had to be counted and shown to the user.

What problem solving techniques did you use?
There were a number of different challenges that had to be overcome, for example we had to pull in information from the web page and sorting out said different information. Somtimes I would draw out the different interactions to establish what the concept was. Rereading the issue to understand what exactly was required to forfill the challenge. Look at the information that was being exchanged between elements of the code and so forth. I was fortunate using these techniques that the problem could be solved.

How did you feel throughout the process?
This was a challenging week, the reading in and extracting the information using JavaScript was difficult to understand. There was a lot of print out of what had been done to understand what infact I was doing. In a way I was lucky to get as far as I did with the work.

What did you learn?
Proabably just to relax and enjoy the work. Time is always a factor, however I have come very far in the process and have learnt a lot. I think that I was quite prepared for the week of work which was a real advantage.

    Describe what these functions do in your own words:
  • .map(); Say you have a list of numbers saved in some code, and you want to carry out some processes with this code. Using the Map function you can collectively carry out functions on these numbers without effecting the origional numbers.
  • .filter(); Again if you have a list of numbers and you wish to carry out some processes, say find which numbers are above or below a certain value. You can use this function to establish which numbers in a list are above or below said value.
  • .reduce(); This function allows a list of numbers to be reduced to a specific number. This function will not execute if some of the values are not numbers. Again used in a list of numbers.
