Technical Blog Part 1

HTML

HTML or Hypertext Markup Language is the language used for web browsers to establish the structure of Web pages. In essence HTML allows a page to be broken down into its most basic components and be able to be read by a web browser. Content is encapulated is this language so as it can be ledgable for a Web Browser.

CSS

CSS or Cascading Style Sheets is a way in which information can be displayed on a particular webpage. CSS allows web developers to create a meaningful and presentable style onto a web page to carry out a specific function

DOM

DOM or the Document Object Model is a way in which a page can be structured in to sections and subscetions. Say for example a begining, middle and end. These areas can have unique identifiers associated with these areas that can also allow for manipulation

What is meant by boxifying design?

My understanding of Boxifying design is when Design/User Experience Consultant or Graphics Designer hands over a design of a proposed website or page thereof. At this point the Developer will break down the page into its basic components through layering boxes over the design to isolate the challenges of the design of the page. From here the Developer has a plan of attack in developing the page.

What is the box model?

All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.
