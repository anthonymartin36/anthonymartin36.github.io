T3 Design to Web

What a responsive site is, and why responsiveness is important.
A responsive site is when a website is viewable from multiple devices say for example a mobile device or a large computer screen. Responsiveness is important because, the we is now being viewed by multiple devices from a broad audience. To make information and services available today on the web, sites must be responsive.

What mobile first design is, and why it's important.
The best description I have read, "The mobile-first approach is exactly as it sounds: designing for the smallest screen and working your way up. It is one of the best strategies to create either a responsive or adaptive design". It is important to know because how we interact with the small screen is as important as how we access information on other devices. Interaction with smaller screens has its restrictions in terms of styles, images and interactions, keeping simplicity at the core of the experience can help.

What frameworks are, and their pros and cons.
The main framework that was used fro this work was the Skeleton . There are a number of different frameworks, the main one that has been worked with here is Skeleton.

Pros Cons
The framework was easy to plugin and use. The framework needed considerable time to come up to speed with how to use it.
A lot of the simple fundementals are already in place for you to start. As an newbie, there were some complications about understanding how to implement some more specifics.

What a wireframe is and why we use it.
A wireframe is effectively a template from which to start developing a web page from. The process is to create an underlying well formatted page structure to support the page content and site navagation that is to be displayed.
